# 🚑 Emergency Risk Prediction System

## 📌 Overview

A machine learning project that predicts patient risk levels in emergency situations using vital signs, medical conditions, and symptoms.

The system classifies patients into:

* High Risk
* Medium Risk
* Low Risk

---
## 📊 Data

Synthetic dataset generated using realistic medical distributions:

* Age, Heart Rate, Blood Pressure
* Temperature, Glucose
* Diabetes status
* Symptoms

---
## 🤖 Model

* Random Forest Classifier (main model)

---
## 📈 Results

* High classification performance across all risk levels
* Key risk factors:

  * Glucose
  * Blood Pressure
  * Heart Rate
  * Critical symptoms

---
## 🧠 Features

* Data preprocessing & feature engineering
* Model training and evaluation
* Feature importance analysis

---
## 🖥 Interactive Demo

* Built using `ipywidgets`
